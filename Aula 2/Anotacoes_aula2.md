# Desenvolva suas habilidades com um jogo de advinhação

O `prompt` é um comando semelhante ao `alert`, mas que permite que o usuário insira algum dado.

`=` -> atribuição
`==` -> comparação
`!=` -> diferente

OBS: Comentários em js: `//`

O `else` engloba todas as possibilidades que não estão incluídas no if

### Gerar um número aleatório:
~~~javascript
parseInt(Math.random() * 1001)
// parseInt = analise como inteiro
// Math.random() = escolhe um número aleatório entre 0 e 0.9999999 (exclui o 1)
// Multiplica por 1001 para fazer um número de 0 a 1000, incluindo os dois extremos
~~~
